About the Project

Our mission

Developing and maintaining a quality daily devotional service is still a complex process for churches and religious organizations.

We know that creating exclusive and consistent devotional content ends up competing with basic tasks such as providing verses, sermons, commentaries, social media content and so many other daily tasks.

We believe we can offer many of these services, focused on daily devotionals, for free, with professional quality and centered on God's Word.

Multi-language

Support for Portuguese and English, with complete translations of all devotionals, authors and themes.

Documentation

RESTful API with clear documentation and intuitive endpoints for easy integration.

Admin Panel

Modern interface to manage devotionals, authors and themes with rich text editor.

Biblical References

Each devotional can contain multiple structured biblical references.
